1. Установите зависимости:

   ```bash
   npm install
   ```

2. Соберите приложение в режиме продакшена:

   ```bash
   npm run build:prod
   ```

3. Запустите приложение:

   ```bash
   npx serve -s dist --single
   ```

4. Откройте браузер и перейдите по адресу:

   ```
   http://localhost:3000
   ```

5. Войдите в приложение, используя следующие креды:

   - **Логин:** `deno`
   - **Пароль:** `deno`
